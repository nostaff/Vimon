Button

其他 / Button按钮组件

基础的按钮组件, 可以设置大小, 形状等, 包括和Icon组件的组合.

如何引入

// 引入
import { Button } from 'vimo
// 安装
Vue.component(Button.name, Button)
// 或者
export default{
  components: {
   Button
 }
}
源码:
  • button/index.js , line 1
